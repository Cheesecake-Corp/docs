The Cheesecake Corp API is organized around REST. It uses standard HTTP verbs, returns JSON, and requires authentication.

## Base URL

```text
https://api.cheesecakecorp.org/v1
```

## Errors

| Status | Meaning |
| --- | --- |
| `200` | OK |
| `400` | Bad request |
| `401` | Unauthorized |
| `404` | Not found |
| `500` | Server error |
